前言

一夜之间，全球最强 AI 模型易主。大模型行业再次经历变革。🔄

Claude 3 发布

爱技术分享网获悉，3月5日凌晨，OpenAI的竞争对手、得到谷歌和亚马逊投资支持的Anthropic公司发布了最新的Claude 3系列模型，其多个领域的性能超越了OpenAI的GPT-4。

Claude 3系列模型包括Haiku（中杯）、Sonnet（大杯）与Opus（超大杯）三款，能力层次递增。在推理、数学、编码、多语言理解和视觉等领域的20项测试中，Claude 3 Opus在14项性能指标上超越了GPT-4和谷歌Gemini等一系列模型。🚀

Claude 3 的能力

Anthropic 表示，Claude 3 Opus 拥有相当于人类本科生水平的知识，并已成为全球最强大的AI大模型。

此外，Claude首次公布了对多模态能力的支持（Opus 版本的 MMMU 得分为 59.4%，超过 GPT-4V，与Gemini 1.0 Ultra持平）。现在，用户可以上传照片、图表、文档等非结构化数据，让 AI 进行分析和解答。🔍

目前，Claude 3 Opus和Sonnet已经在claude.ai以及全球159个国家和地区的Claude API上线。而Haiku模型也即将推出。

  • 有网友称，Claude 3来了，OpenAI准备拿什么来回应？GPT-5,Altman，Sora将何时发布？

  • 单词处理能力是ChatGPT近50倍，Claude 3究竟有多强

这家刚刚超越ChatGPT-4的公司成立不到3年，由OpenAI前研究副总裁和GPT-3论文的主要作者创立。🌟

Anthropic的成立是因为创始团队对OpenAI的发展方向有所分歧，担心公司可能会偏离其最初的愿景。在过去一年，Anthropic完成了五轮融资，总额约73亿美元。

Anthropic联合创始人透露，公司有60至80人参与核心模型的开发，120至150人参与技术研发。Claude模型的最新迭代由约30至35人的团队直接参与。

  • Anthropic最新发布的Claude 3系列模型，在多个性能指标上都非常强大，与GPT-4相抗衡。

GPT-5即将到来？

随着Claude 3的发布，行业内对AI的关注再度升温。马斯克对这一成就表示感动，而刚离职OpenAI的Logan对Anthropic团队表示祝贺。

但也有人质疑Claude对中文支持的程度及其价格策略。据悉，GPT-4 Turbo的收费标准与Claude 3 Opus相比，后者要昂贵得多。

OpenAI最近的一份泄露文件显示，公司计划在2027年前开发出人类级别的通用人工智能。CEO奥特曼表示，OpenAI开发出了超乎想象的新技术，预示着模型能力的巨大飞跃。🌐

图灵奖得主LeCun对大模型公司可能对AI模型行业的影响表示担忧。如果GPT-5即将发布，我们可能即将见证AI领域的一场新热战，更接近AGI时代。🔮

Claude 3 如何注册和使用

在官网注册账号
打开claude官网https://claude.ai/，输入你的邮箱进行注册，你可以使用国内邮箱，或者直接使用google邮箱注册,推荐使用google邮箱，方便快捷。但是没有google邮箱也不要慌，国内邮箱也可以注册：

登录邮箱获取验证码，输入验证码，

验证通过，下一步输入手机号码进行验证

划重点，很多人卡在这一步，因为没有国外的手机号码，这里推荐注册Wildcard,可以获得3次海外手机验证码。

使用Wildcard获取海外手机验证码

这里介绍使用手机上的浏览器获取海外手机验证码的方法

  • 注册完wildcard之后，点击我的服务，找到海外手机号，点击使用，会弹出下面的界面，点击申请手机号。

  • 这里可以使用手机号注册Claude2和OpenAI的账号，我们选择Claude2.

  • 下面是一个提示，点继续。

  • 接着就会显示本次申请到的海外手机号，和这个手机号所属的国家。

  • 接下来回到Claude的注册画面，，以上面的手机号为例，选择国家为英国，也就是United Kingdom,不要输入+44，输入那串黑色的数字，，点击Send Verification Code。注意，你有可能会像下图一样，遇到Phone number already used.Please try a different phone number.的错误，也就是说，电话号码已经被使用，建议换一个号码，不要慌我们看下面的解决办法。

  • 我们回到wildcard的申请手机号码页面，点击申请新号码，马上就会生成一个新号码

  • 把新号码填到Claude的注册画面里，点击Send Verification Code。

  • 没有提示手机号码被占用，那就稍等片刻，手机上wildcard的画面上就会收到短信验证码，这个验证码也会以短信的形式同步发送到你注册wildcard所使用的手机号上。

  • 把验证码输入上面的框，勾选已满18岁的框，点击Verify&Create Account，来验证并创建账号。

验证通过就可以开始使用Claude了。
整体使用跟国内的大模型界面比较类似，跟chatGPT还是有一些区别的。
